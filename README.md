# AI秘書×ゲーム感覚タスク管理アプリ 設計書

## 1. 概要
本アプリケーションは、日々の作業内容やタスクの記録を「チャット形式」で気軽に行い、それをAIが自動的に時間集計・整理するタスク管理ツールである。また、継続的な記録を促進するため、ゲーミフィケーション（レベルアップや報酬メッセージ）を取り入れている。

## 2. ターゲット・解決する課題
- **ターゲット**: 日報の作成や、何にどれだけ時間を使ったかの記録にストレスを感じている全社員。
- **解決する課題**: 
  - 「1日の終わりに何をしていたか思い出せない」という忘却の防止。
  - 入力の手間（フォームへの細かな入力）を、チャットによる自然言語入力で解消。

## 3. 主要機能一覧
1. **チャット入力機能**: 「今から会議」「データ入力おわり」など、ユーザーが現在の状況を自由に入力できる。
2. **時間自動計算**: 入力されたタイムスタンプを元に、それぞれの作業にかかった時間を自動で算出する。
3. **レベルアップシステム**: タスク完了や作業時間の記録に応じて「経験値」が溜まり、ユーザーの「レベル」が上昇する。
4. **AI応援メッセージ**: レベルアップ時や作業終了時に、モチベーションを高めるポジティブなフィードバックを表示する。
5. **履歴表示（ダッシュボード）**: 今日1日、何に何分使ったかをグラフやタイムラインで振り返る機能。

## 4. 画面レイアウト（構成案）
画面はシンプルに1画面（シングルページ）で構成する。
- **左側（メインエリア）**: チャットタイムライン。下部にテキスト入力欄と「送信」ボタンを配置。
- **右側（ステータスエリア）**: 
  - 上部：現在のユーザーレベル、経験値ゲージ、AIからの今日の一言。
  - 下部：本日の作業時間の自動集計結果（例：「会議: 60分」「開発: 120分」）。

## 5. 技術スタック・データ保存
- **フロントエンド**: HTML5 / CSS3 / JavaScript (Vanilla JS)
- **インフラ**: サーバー不要（GitHub Pages などの無料静的ホスティングを利用）
- **データ保存**: 個人情報や機密情報の漏洩を防ぐため、データはすべてユーザーのブラウザ内（LocalStorage）にのみ完全匿名で保存する。

---
*※本設計書は、社内AI活用コンペの要項に準拠し、作成者を特定できる情報を一切含まない匿名形式で作成されています。*
